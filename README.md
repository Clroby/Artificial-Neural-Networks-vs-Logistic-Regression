## Project Overview
Applied bioengineering machine learning study investigating classical logistic regression and neural network–based models for heart disease classification using clinical data.

## Full Technical Report
The complete technical report for this project including abstract, background, methods, architecture, tuning, results, and discussion is available here:

[Logistic Regression vs Artificial Neural Networks for Heart Disease Prediction](report/ANN vs Log Reg Report.pdf)

## Methods
- Logistic Regression (baseline model)
- Artificial Neural Network (ANN)
- Hyperparameter tuning
- Performance evaluation and comparison

- `v1.m` – model training and pipeline
- `v1eval.m` – model evaluation and metrics
- `v1parametertuning.m` – hyperparameter tuning
